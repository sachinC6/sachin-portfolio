# Project Images

Upload your project screenshots here to replace the placeholder images in the portfolio.

## Image Requirements:

- **Dimensions:** 400x600px (portrait orientation)
- **Format:** PNG, JPG, or WebP
- **File size:** Under 500KB for optimal loading
- **Naming:** Use descriptive names (e.g., `weather-station.png`, `fpga-processor.png`)

## How to Add Your Images:

1. Upload 7 project screenshots to this folder
2. Name them: `project1.png`, `project2.png`, etc.
3. Update `index.html` carousel section (lines ~138-144):

```html
<li style="background-image: url('assets/projects/project1.png')"></li>
<li style="background-image: url('assets/projects/project2.png')"></li>
<!-- ... and so on -->
```

## Current Placeholder Images:

The portfolio currently uses CodePen placeholder images. These will be replaced with your actual project screenshots.

## Tips:

- Use high-quality screenshots of your projects
- Show the most interesting visual aspect of each project
- Ensure images are well-lit and clear
- Consider adding text overlay or project logo in the image itself
